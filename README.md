
# Alien Strike

A simple uI game developped using python and pygame module of python


## Run Locally

Clone the project

```bash
  git clonehttps://github.com/ggourab289/Alien-Strike/
```

Go to the project directory

```bash
  cd Alien-Strike
  cd game
```

Install dependencies

```bash
  pip install pygame
```

Start the game

```bash
  python main.py
```
